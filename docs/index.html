Get notified when your stock gets cheap

Stocks Notifier is a software which tracks real-time stock prices and sends notifications when price of stock is lower than or equal to threshold value.

Follow the below steps to download and run this locally.

Installation (MacOS)

Download executable file

Download stocks.json and list the stocks and threshold value you want.

Check Yahoo Finance for naming convention of stocks.

Copy and run the below command to try it out.

./stocks-notifier .

Pass the directory where stocks.json is located.

Why?

Why not? For fun. I wanted to get notification when a stock's price gets lower than a value, without having to sign up on a software using my email or phone number. Similarly I didn't want to get ads and wanted something which didn't use lot of CPU and memory.
